## 説明文

Video to Live Photo は、手持ちの動画から Live Photo を作成し、iPhone の写真ライブラリに保存するためのシンプルなアプリです。

保存した Live Photo は、iPhone 標準の「設定」→「壁紙」からロック画面の壁紙として選べます。壁紙として設定する手順はアプリ内でわかりやすく案内します。

### 特徴
- 動画から短いクリップを切り出して Live Photo として保存
- 作成した Live Photo のプレビュー表示
- 壁紙設定手順の丁寧なガイド（設定 App への自動遷移はありません）
- 縦向きの壁紙向けレイアウトに最適化

### 料金
- 無料: 標準画質で保存できます（広告あり）
- 高画質お試し: 短い広告を視聴すると、1回だけ高画質で保存できます
- 買い切りで高画質解放: 一度の購入で高画質書き出しがいつでも使えるようになり、広告も非表示になります
  - 商品ID: `com.gen.videotolivephoto.premium.hq_unlock`
  - 非消耗型（一度購入すると再購入は不要）

### 注意事項
- 本アプリは壁紙を自動で設定することはできません（iOS の制約により、一般アプリではロック画面の壁紙の自動設定はできません）
- 壁紙として使う場合は、保存後に iPhone 標準の「設定」App から選択してください

## App Store レビュワー向けノート（審査メモ草案）

- 本アプリは動画から Live Photo を作成して写真ライブラリに保存する機能のみを提供します。
- 壁紙の自動設定はサポートしません。ロック画面の壁紙設定は iPhone 標準の「設定」App からユーザー自身が行います。
- 非消耗型 IAP `com.gen.videotolivephoto.premium.hq_unlock` は、購入すると高画質書き出しが恒久解放され、広告が完全に非表示になります。一度購入すれば、再インストール後も「購入を復元」から再利用できます。
- 広告はリワード広告のみで、ユーザーが明示的にタップした場合のみ表示されます。変換の進捗画面を強制的に中断して広告を表示することはありません。
- 写真ライブラリへのアクセスは `.addOnly` 権限で要求しています。閲覧権限は要求しません。
- プライベート API は使用していません。
- テストアカウント不要（機能はアプリ内で完結します）。
- 技術スタック: React Native (Expo SDK 51) + TypeScript。Live Photo の MOV/JPEG タグ付けと `PHAssetCreationRequest` は、独自に実装した Expo Module（Swift）で行っています。ビルドと App Store Connect への提出は EAS Build / EAS Submit（クラウド）で行っています。

## スクリーンショット構成案（6枚）

1. **Home**: 大きな「動画を選ぶ」ボタン中心のランディング
2. **Picker**: 写真ライブラリから動画を選んでいる様子
3. **Trim**: 動画の長さを短いクリップに調整しているところ
4. **Export Options**: 「標準画質」「広告で高画質お試し」「買い切り解放」の3択
5. **Preview**: Live Photo のプレビュー表示と「長押しで再生」の吹き出し
6. **Success Guide**: 保存完了と「設定 → 壁紙 → 新しい壁紙を追加 → 写真 → Live Photo」の4ステップガイド

各スクリーンショットのキャプションには、壁紙を自動で設定する旨の表現を**絶対に使わない**こと（詳細は `CLAUDE.md §3` の禁止表現リストを参照）。
